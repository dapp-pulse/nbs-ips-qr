# NBS IPS QR

Java library for generating NBS IPS QR codes according to the [National Bank of Serbia IPS QR specification](https://ips.nbs.rs/sr_lat).

## Usage

**Maven**
```xml
<dependency>
    <groupId>com.dappulse</groupId>
    <artifactId>nbs-ips-qr</artifactId>
    <version>VERSION</version>
</dependency>
```

**Gradle**
```kotlin
implementation("com.dappulse:nbs-ips-qr:VERSION")
```


## Build

**Prerequisites**
- Java JDK 21+
- Apache Maven 3.8+

Build and install to local repository:

```bash
mvn clean install
```

Run tests only:

```bash
mvn test
```
